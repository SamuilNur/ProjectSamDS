Financial Analytics & Profit Forecasting
Анализ финансовых данных компании и прогнозирование прибыли с помощью машинного обучения. Полный цикл: очистка данных → EDA → бизнес-аналитика → ML-модели → прогноз.

Стек
Python · Pandas · NumPy · Matplotlib · Seaborn · Scikit-learn · Google Colab

Структура

├── Financial_Analysis.ipynb   # Основной ноутбук
├── models.py                  # Модели, тюнинг, прогноз
├── Financials.csv             # Датасет (700 транзакций, 5 стран, 6 продуктов)
└── README.md
Что сделано
Очистка — парсинг валют, обработка дат, проверка на пропуски
EDA — распределения, корреляции, выбросы, 20+ визуализаций
Бизнес-анализ — маржинальность по сегментам, влияние скидок, убыточные сделки
ML — сравнение 5 моделей, кросс-валидация, тюнинг Gradient Boosting через GridSearchCV
Прогноз — месячная прибыль на 6 месяцев с доверительным интервалом
Результат
Лучшая модель: Gradient Boosting (R² = 0.99, MAPE < 10%)

Прогноз на 6 месяцев: ~$8M суммарно (~$1.34M/мес.)

Прогноз построен на 16 месяцах данных — для надёжных предсказаний нужно 24+
Ключевые выводы
Government — самый прибыльный сегмент
Высокие и средние скидки снижают маржу и приводят к убыткам (особенно Enterprise)
Paseo — лидер продаж, но не самый маржинальный
Главные драйверы прибыли: COGS, Sales, Gross Sales, Units Sold
