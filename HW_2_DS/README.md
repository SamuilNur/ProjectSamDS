Датасет брал с datahub.io - Индекс потребительских цен США и инфляция (CPI)

В этой домашней работе реализованы следующие пункты:

- Созданы модули для работы с данными:
  - data_loader.py – загрузка данных (CSV, JSON, API)
  - data_processing.py – проверка и обработка пропусков, конвертация дат
  - visualization.py – построение графиков: линейные, гистограммы, scatter
- Загружен датасет CPI (Consumer Price Index, США)
- Проверены пропуски и при необходимости заполнены
- Построены графики для анализа данных (Index и Inflation)
- Код организован по модулям для удобства и повторного использования
