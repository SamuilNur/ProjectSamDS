Цель: сравнить 5 регрессоров на Students Performance Dataset, предсказывая GPA.
GPA (Grade Point Average) — это средний балл успеваемости студента.


Модели: Linear Regression, Ridge, Decision Tree, Random Forest, Extra Trees.

Провёл:

загрузку и препроцессинг данных,

обучение всех моделей,

сравнение метрик (MAE, RMSE, R²),

настройку гиперпараметров для топ‑3 моделей,

много визуализаций предсказаний и ошибок. И визуализаций итогового предсказания моделями.

Результаты:

Лучшие модели: Linear Regression и Ridge (R² ≈ 0.956, MAE ≈ 0.151)

Деревья (Extra Trees, Random Forest) чуть хуже, Decision Tree уступает заметно.

Настройка гиперпараметров дала минимальные улучшения, данные хорошо описываются линейной моделью.

Вывод: GPA линейно зависит от признаков, поэтому Linear Regression/Ridge — оптимальный выбор для этого датасета.
