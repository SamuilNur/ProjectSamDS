Прогнозирование времени доставки в логистике
Обзор проекта

Данный проект посвящён прогнозированию времени доставки в логистических операциях на основе исторических данных о перевозках. Основной целью работы является сравнение различных подходов прогнозирования и определение наиболее эффективных методов для предсказания времени доставки в условиях высокой вариативности логистических процессов.

В рамках исследования были рассмотрены два основных подхода:

классические модели временных рядов

методы машинного обучения

Сравнение этих подходов позволяет оценить, какие модели лучше справляются с задачей прогнозирования в условиях нестабильных и шумных данных.

Данные

В проекте используется датасет, содержащий информацию о логистических перевозках.

Данные включают различные параметры, связанные с процессом доставки:

маршрут перевозки

этапы транспортировки (transport legs)

временные метки

операционные параметры перевозки

Целевая переменная:

время доставки (Delivery Time)

Особенностью датасета является:

высокая вариативность значений

наличие значительных выбросов

резкие скачки времени доставки

Такие характеристики усложняют задачу прогнозирования и требуют применения более гибких моделей.

Предобработка данных

Перед построением моделей были выполнены следующие этапы подготовки данных:

очистка и структурирование данных

обработка пропущенных значений

подготовка признаков

разделение данных на обучающую (train) и тестовую (test) выборки

подготовка временного ряда для моделей прогнозирования

Используемые модели

В работе были применены два класса моделей.

1. Модели временных рядов

Для анализа временной структуры данных была использована модель:

ARIMA

Модель основана на анализе автокорреляций временного ряда и используется для прогнозирования значений на основе предыдущих наблюдений.

2. Модели машинного обучения

Также были применены несколько регрессионных алгоритмов машинного обучения:

Linear Regression

Decision Tree

Random Forest

Support Vector Regression (SVR)

Gradient Boosting

В отличие от моделей временных рядов, данные алгоритмы могут использовать несколько признаков одновременно и способны моделировать более сложные зависимости между переменными.

Оценка моделей

Для анализа результатов использовались:

численные метрики качества моделей

визуализация прогнозов

сравнение предсказаний различных моделей

Были построены следующие типы визуализаций:

графики Train / Test / Forecast

сравнение реальных и прогнозируемых значений

графики предсказаний моделей машинного обучения

Для визуального анализа были выбраны три репрезентативные модели:

Linear Regression

Random Forest

Gradient Boosting

Эти модели представляют различные классы алгоритмов машинного обучения:

Тип модели	Представитель
линейные модели	Linear Regression
деревья решений	Random Forest
ансамблевые методы boosting	Gradient Boosting

Остальные алгоритмы (Decision Tree и SVR) были включены в сравнительный анализ по метрикам, но не рассматривались отдельно на графиках, поскольку их поведение не добавляло принципиально новой информации к визуальному сравнению моделей.

Результаты

Анализ показал несколько важных особенностей данных и поведения моделей.

1. Высокая вариативность данных

Данные о времени доставки содержат большое количество выбросов и резких скачков значений. Это характерно для логистических процессов, где задержки могут возникать по различным причинам:

погодные условия

задержки на маршруте

проблемы на складах

операционные сбои

Такая структура данных значительно усложняет задачу прогнозирования.

2. Поведение модели ARIMA

Модель ARIMA показала сглаженный прогноз и не смогла корректно предсказывать резкие изменения времени доставки.

Это связано с тем, что модель ориентируется на временные зависимости и предполагает относительно стабильную структуру временного ряда. В рассматриваемом датасете такие зависимости выражены слабо.

3. Поведение моделей машинного обучения

Модели машинного обучения продемонстрировали более гибкое поведение, поскольку используют дополнительные признаки данных.

Особенно хорошо себя показали ансамблевые методы:

Random Forest

Gradient Boosting

Эти алгоритмы лучше адаптируются к сложной структуре логистических данных и способны учитывать нелинейные зависимости.

Итоговые выводы

По результатам исследования можно сделать следующие выводы:

данные о времени доставки обладают высокой вариативностью и содержат значительные выбросы

классические модели временных рядов (ARIMA) плохо подходят для прогнозирования таких данных

модели машинного обучения демонстрируют более устойчивые результаты

ансамблевые методы, такие как Random Forest и Gradient Boosting, лучше адаптируются к сложной структуре логистических данных

Таким образом, для задач прогнозирования времени доставки методы машинного обучения оказываются более эффективными, чем классические модели временных рядов.
